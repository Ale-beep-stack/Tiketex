# 🎫 Generador de Tickets de Facturación

Sistema profesional para generar tickets de facturación a partir de facturas PDF (DTE El Salvador).

## ✨ Características

- 📄 **Extracción automática** de datos desde PDF de facturas
- 🎫 **Generación de tickets** en formato térmico (80mm)
- 📦 **Control de inventario** integrado
- 🖨️ **Impresión directa** a impresoras térmicas
- 📊 **Reportes y estadísticas**
- 🔄 **Actualización automática** desde GitHub
- 💾 **Gestión de emisores** múltiples

## 🚀 Instalación

### Descarga e instalación automática:

1. Ve a la página de [Releases](https://github.com/Ale-beep-stack/Tiketex/releases/latest)
2. Descarga el archivo **`GeneradorTickets_Instalador.exe`**
3. Ejecuta el instalador (doble clic)
4. ¡Listo! El programa se instalará automáticamente

El instalador creará:
- ✅ Acceso directo en el escritorio
- ✅ Entrada en el menú inicio
- ✅ Configuración inicial

## 🔄 Actualizaciones

El programa verifica automáticamente si hay nuevas versiones al iniciar. Si hay una actualización disponible:
- Se descarga automáticamente
- Se instala sin intervención del usuario
- El programa se reinicia con la nueva versión

**No necesitas volver a descargar nada manualmente.**

## 📋 Requisitos del Sistema

- Windows 7 o superior
- 100 MB de espacio en disco
- Conexión a internet (para verificar actualizaciones)
- Impresora térmica de 80mm (opcional)

## 🎯 Uso Rápido

1. **Configurar Emisor**: Agrega los datos de tu negocio
2. **Cargar Factura**: Selecciona el PDF de la factura DTE
3. **Generar Ticket**: El ticket se genera automáticamente
4. **Imprimir**: Envía el ticket a tu impresora térmica

## 🛠️ Funcionalidades Avanzadas
